About Dandle

How to Play

Diagnose the neurosurgical case using as few clues as possible. Each case has five clue tiers:

🧾
History — always visible from the start
🔬
Exam — unlocks after 1st wrong guess
🧠
Imaging — unlocks after 2nd
🧪
Labs / Special — unlocks after 3rd
⚕️
Intraop / Pathology — unlocks after 4th
  • You can also manually reveal the next clue at any time (costs an attempt).
  • Correct on the 1st clue = ★★★★★ (five stars). Each extra clue costs a star.
  • Each case links to AANS, CNS, BTF, NASS or other society guidelines.
  • Daily mode: one new case each day, same for everyone. Streak counts!
  • Practice mode: random case from the full library.
  • Category mode: filter by subspecialty and difficulty tier.
